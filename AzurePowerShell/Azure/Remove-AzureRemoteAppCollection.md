# Remove-AzureRemoteAppCollection
## SYNOPSIS
This cmdlet removes a Microsoft Azure RemoteApp collection.

## SYNTAX

```
Remove-AzureRemoteAppCollection [-CollectionName] <String> [-Profile <AzureSMProfile>] [-WhatIf] [-Confirm]
```

## DESCRIPTION
This cmdlet deletes a Microsoft Azure RemoteApp collection.
