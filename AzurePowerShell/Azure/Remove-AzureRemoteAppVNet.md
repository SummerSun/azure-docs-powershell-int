# Remove-AzureRemoteAppVNet
## SYNOPSIS
This cmdlet deletes a Microsoft Azure RemoteApp virtual network.

## SYNTAX

```
Remove-AzureRemoteAppVNet -VNetName <String> [-Profile <AzureSMProfile>]
```

## DESCRIPTION
This cmdlet deletes a Microsoft Azure RemoteApp virtual network.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Remove-AzureRemoteAppVnet -VNetName ContosoVNet
```
