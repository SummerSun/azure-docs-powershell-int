# Get-AzureRemoteAppWorkspace
## SYNOPSIS
This cmdlet retrieves information for a Microsoft Azure RemoteApp workspace.

## SYNTAX

```
Get-AzureRemoteAppWorkspace [-Profile <AzureSMProfile>]
```

## DESCRIPTION
This cmdlet retrieves information for a Microsoft Azure RemoteApp workspace.

