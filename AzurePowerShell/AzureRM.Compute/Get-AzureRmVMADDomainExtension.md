# Get-AzureRmVMADDomainExtension
## SYNOPSIS
Gets information about a AD domain extension.

## SYNTAX

```
Get-AzureRmVMADDomainExtension [-ResourceGroupName] <String> [-VMName] <String> [-Name] <String> [-Status]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>]
```

## DESCRIPTION
The Get-AzureRmVMADDomainExtension cmdlet gets information about the Azure AD Domain extension.
