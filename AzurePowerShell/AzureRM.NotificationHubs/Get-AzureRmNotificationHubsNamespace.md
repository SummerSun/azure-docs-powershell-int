# Get-AzureRmNotificationHubsNamespace
## SYNOPSIS
Gets a list or a particular Namespace description.

## SYNTAX

```
Get-AzureRmNotificationHubsNamespace [[-ResourceGroup] <String>] [[-Namespace] <String>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>]
```

## DESCRIPTION
Get-AzureRmNotificationHubsNamespace cmdlet gets a list of all Namespaces of type NotificationHub under a subscription or specified resource group.
