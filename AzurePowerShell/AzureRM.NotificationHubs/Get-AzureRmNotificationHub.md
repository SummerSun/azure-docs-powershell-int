# Get-AzureRmNotificationHub
## SYNOPSIS
Gets a list or a particular NotificationHub description.

## SYNTAX

```
Get-AzureRmNotificationHub [-ResourceGroup] <String> [-Namespace] <String> [[-NotificationHub] <String>]
 [-InformationAction <ActionPreference>] [-InformationVariable <String>]
```

## DESCRIPTION
Get-AzureRmNotificationHub cmdlet gets a list of all NotificationHubs under a Namespace or a specified NotificationHub within the namespace.
