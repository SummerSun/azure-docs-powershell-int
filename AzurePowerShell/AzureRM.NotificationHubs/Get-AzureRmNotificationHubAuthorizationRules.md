# Get-AzureRmNotificationHubAuthorizationRules
## SYNOPSIS
Gets a list or a particular Authorization Rule description associated with the specified NotificationHub.

## SYNTAX

```
Get-AzureRmNotificationHubAuthorizationRules [-ResourceGroup] <String> [-Namespace] <String>
 [-NotificationHub] <String> [[-AuthorizationRule] <String>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>]
```

## DESCRIPTION
Get-AzureRmNotificationHubAuthorizationRules cmdlet gets a list of all the Authorization Rules descriptions or a specified Authorization Rule associated with the NotificationHub specified.
