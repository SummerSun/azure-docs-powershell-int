# Remove-AzureRmNotificationHubsNamespace
## SYNOPSIS
Removes an existing Namespace.

## SYNTAX

```
Remove-AzureRmNotificationHubsNamespace [-ResourceGroup] <String> [-Namespace] <String>
 [-InformationAction <ActionPreference>] [-InformationVariable <String>]
```

## DESCRIPTION
Remove-AzureRmNotificationHub cmdlet removes existing Namespace of type NotificationHub.
