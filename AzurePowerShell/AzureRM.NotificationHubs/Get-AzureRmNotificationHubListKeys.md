# Get-AzureRmNotificationHubListKeys
## SYNOPSIS
Gets the Primary and Secondary ConnectionString of a particular NotificationHub Authorization Rule.

## SYNTAX

```
Get-AzureRmNotificationHubListKeys [-ResourceGroup] <String> [-Namespace] <String> [-NotificationHub] <String>
 [-AuthorizationRule] <String> [-InformationAction <ActionPreference>] [-InformationVariable <String>]
```

## DESCRIPTION
Get-AzureRmNotificationHubListKeys cmdlet gets the Primary and Secondary ConnectionString of a particular NotificationHub Authorization Rule.
