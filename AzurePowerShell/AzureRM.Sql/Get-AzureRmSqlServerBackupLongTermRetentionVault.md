# Get-AzureRmSqlServerBackupLongTermRetentionVault
## SYNOPSIS
Gets a server long term retention vault.

## SYNTAX

```
Get-AzureRmSqlServerBackupLongTermRetentionVault [-ServerName] <String> [-ResourceGroupName] <String>
 [-InformationAction <ActionPreference>] [-InformationVariable <String>]
```

## DESCRIPTION
Gets the long term retention vault registered to this server. 
This is an Azure Backup resource used to store backup data.
