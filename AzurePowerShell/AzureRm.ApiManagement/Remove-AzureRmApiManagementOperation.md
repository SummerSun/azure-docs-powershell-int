# Remove-AzureRmApiManagementOperation
## SYNOPSIS
Remove existing operation.

## SYNTAX

```
Remove-AzureRmApiManagementOperation -Context <PsApiManagementContext> -ApiId <String> -OperationId <String>
 [-PassThru] [-InformationAction <ActionPreference>] [-InformationVariable <String>] [-WhatIf] [-Confirm]
```

## DESCRIPTION
Remove existing operation.

## EXAMPLES

### --------------------------  Example 1  --------------------------
@{paragraph=PS C:\\\>}

```
Remove-AzureRmApiManagementOperation -Context $apimContext -ApiId 0123456789 -OperationId 9876543210 -Force
```

Remove existing API Operation.
