# Get-AzureRmSiteRecoveryStorageClassification
## SYNOPSIS
Retrieves Azure Site Recovery Storage classifications.

## SYNTAX

```

### ByName
```
Get-AzureRmSiteRecoveryStorageClassification -Name <String>
```

### ByFriendlyName
```

```

### ByObject
```
