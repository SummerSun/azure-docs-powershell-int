# Remove-AzureStorageShareStoredAccessPolicy
## SYNOPSIS
Remove a Stored Access Policy from an Azure Storage share.

## SYNTAX

```
Remove-AzureStorageShareStoredAccessPolicy [-ShareName] <String> [-Policy] <String> [-PassThru]
 [-Context <AzureStorageContext>] [-ServerTimeoutPerRequest <Int32>] [-ClientTimeoutPerRequest <Int32>]
 [-ConcurrentTaskCount <Int32>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
 [-WhatIf] [-Confirm]
```

## DESCRIPTION
Remove a Stored Access Policy from an Azure Storage share.

## EXAMPLES

### --------------------------  Remove a stored access policy from an Azure Storage share  --------------------------
@{paragraph=PS C:\\\>}

```
PS C:\> Remove-AzureStorageShareStoredAccessPolicy -ShareName test -Policy testPolicy
```
