# New-AzureStorageShare
## SYNOPSIS
Create a new Azure Storage file share.

## SYNTAX

```
New-AzureStorageShare [-Name] <String> [-Context <AzureStorageContext>] [-ServerTimeoutPerRequest <Int32>]
 [-ClientTimeoutPerRequest <Int32>] [-ConcurrentTaskCount <Int32>] [-InformationAction <ActionPreference>]
 [-InformationVariable <String>] [-PipelineVariable <String>]
```

## DESCRIPTION
Create a new Azure Storage file share.

## EXAMPLES

### --------------------------  Example 1  --------------------------
@{paragraph=PS C:\\\>}

```
PS C:\> New-AzureStorageShare -Name sample
```

This example creates a file share named sample.
