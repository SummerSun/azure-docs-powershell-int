# New-AzureRmServerManagementGateway
## SYNOPSIS
Creates a new Server Management Gateway

```
 [-AutoUpgrade] [-Tags <Hashtable>] [-InformationAction <ActionPreference>] [-InformationVariable <String>]
```

## DESCRIPTION
