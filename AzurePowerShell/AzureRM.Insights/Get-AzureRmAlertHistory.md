# Get-AzureRmAlertHistory
## SYNOPSIS
Retrieves the history of alerts.

## SYNTAX

```
Get-AzureRmAlertHistory [-ResourceId <String>] [-StartTime <DateTime>] [-EndTime <DateTime>] [-Status <String>]
 [-Caller <String>] [-DetailedOutput]
```

## DESCRIPTION
Retrieves the history of alerts as they get enabled, disabled, fire, resolve, etc.
