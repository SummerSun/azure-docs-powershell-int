# Get-AzureRmMetricDefinition
## SYNOPSIS
Retrieves the metric definitions.

## SYNTAX

```
Get-AzureRmMetricDefinition [-ResourceId] <String> [-MetricNames <String[]>] [-DetailedOutput]
```

## DESCRIPTION
Retrieves the metric definitions.
