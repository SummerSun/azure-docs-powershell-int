# Get-AzureRmAutoscaleHistory
## SYNOPSIS
Retrieves the history of autoscale.

## SYNTAX

```
Get-AzureRmAutoscaleHistory [-ResourceId <String>] [-StartTime <DateTime>] [-EndTime <DateTime>]
 [-Status <String>] [-Caller <String>] [-DetailedOutput]
```

## DESCRIPTION
Retrieves a list of events related to an autoscale setting.

## EXAMPLES

### --------------------------  Example 1: Get all events associated to the subscription  --------------------------
@{paragraph=PS C:\\\>}

```
Get-AzureRmAutoscaleHistory -Star 2015-02-09T18:35:00 -end 2015-02-09T18:40:00 -det
```

This command retrieves all the autoscale-related events associated to the current subscription

### --------------------------  Example 2: GetAutoscaleHistory for a particular resource  --------------------------
